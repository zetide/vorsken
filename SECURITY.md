# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.2.x   | ✅ Yes    |
| < 0.2   | ❌ No     |

## Reporting a Vulnerability

Please report security vulnerabilities to **security@zetide.com**.

You can also open a [GitHub Security Advisory](https://github.com/zetide/vorsken/security/advisories/new).

Do **not** open a public GitHub Issue for security reports.

We will respond within 72 hours and aim to release a patch within 14 days for confirmed vulnerabilities.
